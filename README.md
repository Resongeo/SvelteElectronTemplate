# How to use the template
#### You can use npm if you like but yarn is advised
### Clone the Project
```bash
$ git clone https://github.com/Resongeo/SvelteElectronTemplate
```
### Go into cloned directory
``` bash
$ cd SvelteElectronTest
```
### Install dependencies
``` bash
$ yarn install
```
### Run the app
```bash
$ yarn electron-dev
```
### Package the app
```bash
$ yarn electron-pack
```
